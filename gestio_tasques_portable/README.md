# Gestió de Tasques - PHP MVC + Docker

## Descripció

Aplicació web de gestió de tasques desenvolupada amb PHP natiu i patró MVC.

El projecte és portable i es pot executar amb Docker Compose.

## Requisits

Cal tenir instal·lat:

- Git
- Docker
- Docker Compose

## Instal·lació

Clonar el repositori:

```bash
git clone https://github.com/IkerCaballero/Pr-ctica2-DAW-parejas.git
cd gestio_tasques_portable
docker compose up -d
